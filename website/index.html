Keep Your Mac Awake

A lightweight menu bar utility that prevents idle sleep. No bloat, no subscriptions, no unnecessary features.

Download on GitHub

macOS 15+ · Apple Silicon & Intel

Featherweight

Zero dependencies. Uses native macOS power assertions — the same API the OS itself trusts.

Two Modes

Display & System keeps your screen on. System Only lets the display sleep while blocking idle sleep.

Set & Forget

Launch at login. Remember your last state. Vigil picks up right where you left off.

Lives in Your Menu Bar

No dock icon. No clutter. Just a quiet sentinel in your menu bar.

Vigil is free and open source on GitHub.
